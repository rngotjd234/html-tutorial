1. html 속성

    속성은 html태그에 대한 추가 정보를 제공한다.
  1. html태그는 속성을 추가할 수 있다
  2. 속성은 태그에 대한 추가적인 정보를 제공한다
  3. 속성은 시작태그에 추가한다
  4. 속성은 이름 = "값" 쌍으로 사용한다
  5. 속성값은 인용부호 "값"또는 '값'형식으로 사용 가능하며, 중첩사용도 가능하다(이름="나의 이름은 'ACE'이다")
    예를 들면 html링크는 <a>태그로 정의되고, 링크 주소는 href속성을 사용한다.
.
  • 1.<a href="http://www.w3schools.com">This is a link</a>

    • ※ 속성이름과 속성값은 대/소문자를 구분한다. W3C에서는 속성이름과 속성값으로 소문자사용을 권장한다.

    2. html의 참조 속성

    • class 속성 : 태그에 대한 하나나 그 이상의 클래스(그룹) 이름을 지정
    • id속성 : 태그의 유일한 아이디값을 정의
    • style 속성 : 태그에 CSS 스타일을 인라인으로 정의
    • title속성 : 태그에 대한 정보 정의
